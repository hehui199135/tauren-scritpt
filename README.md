- Helium 是一个自动化库，主要用于网页自动化任务。在 Python 中安装 Helium 可以通过 pip 命令来完成。打开终端或命令提示符，然后输入以下命令：
    ```
    pip install helium
    ```
  如果你在使用 Python 3.x 版本，可能需要使用 pip3 命令：
    ```
    pip3 install helium
    ```
